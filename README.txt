TO COMPILE THE CODE:

1. Install Python 3.x

2. Download the code

3. (Windows/Linux) Move to directory of the code using "cd" command
or
3. (Windows) Move to directory of the code and hit Shift+RMB and select "Open PowerShell window here"

4. python wordsuggestion.py {csv file name here} {search word here}

Example:-
python wordsuggestion.py dictionary.csv hellp
