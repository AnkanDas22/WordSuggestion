TO COMPILE THE CODE:

1. Install Python 3.x if on Windows, Python is preinstalled on Linux

2. Download the code

3. (Windows/Linux) Move to directory of the code using "cd" command
or
3. (Windows) Move to directory of the code and hit Shift+RMB and select "Open PowerShell window here"

4. Linux:
chmod +x wordsuggestion.py
./wordsuggestion.py {csv file name here} {search word here}

Example:-
./wordsuggestion.py dictionary.csv hellp
	            (argument 1)   (argument 2)
or

4. Windows:
python wordsuggestion.py {csv file name here} {search word here}

Example:-
python wordsuggestion.py dictionary.csv hellp
			 (argument 1)   (argument 2)
